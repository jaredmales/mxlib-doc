Create a 2-D Tukey window.

Function to create a 2-D Tukey window.

The width of the window is controlled by alpha. alpha = 0 is a cylinder, alpha=1.0 is the Hann window.

See https://en.wikipedia.org/wiki/Window_function

Parameters
filtis a pre-allocated array of size dim x dim
dimis the size of the array
Nis the diameter of the window, nominally N=dim, but this is not required
alphacontrols the window width.
xcis the desired x center of the window
ycis the desired y center of the window
