struct mx::wfp::idealCoronagraph< _realT >

The Ideal Coronagraph.

A simple toy coronagraph which operates only the pupil plane, subtracting the energy-minimizing wavefront [3]. For an on-axis source with a perfectly flat wavefront this results in perfect extinction. This coronagraph is not physically realizable, but it is often useful for modeling and analysis, and it has been shown that real-world optimized coronagraphs often approach ideal performance [22] [and Males in prep].

Determines how the image is centered.

If 0 (default) it is at 0.5*(wfSz-1), if true it is shifted by 0.5*m_wholePixel in each axis. This is passed to fraunhoferPropagator when it is resized.
