Calculate the 2D integral given a radial profile.

Calculates the integral

\[ 2\pi \int_{min}^{max} p(r) r dr \]

using the Trapezoid rule.

The limits of integration are normally r[0] to r.back() inclusive. If the parameter inczero is set to true, then the region from 0 to r[0] is included, using the value of p[0], where the the integral is there estimate as \( p[0] \pi r[0]^2 \).
