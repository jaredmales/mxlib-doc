Get the Zernike coefficients n,m corrresponding the Noll index j.

Calculates the values of (n,m) for an index j following Noll (1976) [15] See also: http://en.wikipedia.org/wiki/Zernike_polynomials

If j is odd, this returns m <= 0.

Return values
0on success
-1on error (j < 1)
Test:
Verify calculation of Noll nm values from j [test doc]
Parameters
[out]n[out] n the radial index of the Zernike polynomial
[out]m[out] m the azimuthal index of the Zernnike polynomial. m < 0 if j odd.
[in]j[in] j the Noll index, j > 0.

Fill in an Eigen-like array with a Zernike polynomial.

The geometric center of the array, 0.5*(arr.rows()-1), 0.5*(arr.cols()-1), is used as the center. Sets any pixel which is at rad <= r < rad+0.5 pixels to rho = 1, to be consistent with mx::circularPupil

Parameters
[out]arris the allocated array with an Eigen-like interface. The rows() and cols() members are used to size the polynomial.
[in]jis the Noll index of the polynomial
[in]rad[optional] is the desired radius. If rad <= 0, then the maximum radius based on dimensions of m is used.

Fill in an Eigen-like array with the square-normed Fourier transform of a Zernike polynomial.

The array is filled in with the values of |Q(k,phi)|^2 according to Equation (8) of Noll (1976) [15].

Test:
Verify compilation and execution of zernikeQNorm. [test doc]
Returns
0 on success
-1 on error
Template Parameters
arrayTis the Eigen-like array type. Arithmetic will be done in arrayT::Scalar.
Parameters
[out]arr[out] the allocated array. The rows() and cols() members are used to size the transform.
[in]k[in] the normalized spatial frequency magnitude at each pixel
[in]phi[in] the spatial frequency angle at each pixel
[in]j[in] the polynomial index in the Noll convention [15]
