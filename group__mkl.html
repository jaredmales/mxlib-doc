Instructions for installing the Intel Math Kernel Library (MKL)

To install the MKL:

  1. Download the library from Intel. This requires registration and following some links in emails, etc. Unpack it.
  2. Install the MKL library by running the install.sh script. The default paths, etc, should be fine. Be sure to do this with root privileges if you can.

  3. Then add the following line to your profile: 

    source /opt/intel/mkl/bin/mklvars.sh intel64

    This can be done in your ~/.bashrc or ~/.bash_profile or ~/.bash_aliases file (whichever you use for this). Or to set it for all users add it to /etc/profile.d/mkl.sh (create the file).

  4. You will also need to update ld. The system wide way to do this is to create a file called /etc/ld.so.conf.d/mkl.conf with the contents 

    /opt/intel/compilers_and_libraries_2018.3.222/linux/mkl/lib/intel64_lin/

    edited as appropriate for your version.

    Then run 

    $ sudo ldconfig
