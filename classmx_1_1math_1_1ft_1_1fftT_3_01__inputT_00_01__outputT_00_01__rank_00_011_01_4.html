template<typename _inputT, typename _outputT, size_t _rank>
class mx::math::ft::fftT< _inputT, _outputT, _rank, 1 >

Fast Fourier Transforms using the CUDA library.

The CUFFT Templates type resolution system is used to allow the compiler to access the right plan and types for the transforms based on inputT and outputT.

Template Parameters
_inputTis the input type of the transform, can be either real or complex
_outputTis the output type of the transform, can be either real or complex
_rankis the rank of the transform. Limited to 2 for now.
