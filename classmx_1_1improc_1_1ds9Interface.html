An interface to the ds9 image viewer.

Handles spawning the ds9 window, and manages shared memory segments, one for each frame.

Display an image in ds9.

A new ds9 instance is opened if necessary, and a new sharedmemory segment is added if necessary. The image is described by a pointer and its 2 or 3 dimensions.

Return values
0on sucess
-1on an error
Parameters
[in]im[in] the address of the image
[in]dim1[in] the first dimension of the image (in pixels)
[in]dim2[in] the second dimension of the image (in pixels)
[in]dim3[in] the third dimension of the image (in pixels), set to 1 if not a cube.
[in]frame[in] [optional] the number of the new frame to initialize.
Note
frame must be >= 1.
