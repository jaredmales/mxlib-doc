template<class _ccImT>
class mx::improc::imageXCorrFFT< _ccImT >

Find the optimum shift to align two images using the Fourier cross correlation and a peak fit.

The shift is reported in pixels such that if the mxlib imageShift function is used to shift the input image by the negative of the shifts, it will align with the reference.
