Force generation of new screens if true.

Default c'tor Setup the overall atmosphere.

Parameters
[in]pupD[in] New value of _pupD, the size of the wavefront in meters.
[in]wfSz[in] New value of _wfSz, the size of the wavefront in pixels.
[in]buffSz[in] New value of _buffSz, the size of the interpolation buffer to use.
[in]lam0[in] New value of _lambda0, the wavelength at which r_0 is measured.
[in]lam[in] New value of _lambda, the wavelength at which the phase is specified.
[in]subPist[in] New value of _subPiston, whether or not piston is subtracted from the wavefront phase.
[in]subTip[in] New value of _subTipTilt, whether or not tip/tilt is subtracted from wavefront phase.
